# Sk-Modernist Font Files

Place the Sk-Modernist font files in this directory.

## Required Files

- `Sk-Modernist-Regular.woff2` (weight: 400)
- `Sk-Modernist-Bold.woff2` (weight: 700)

## Optional Files (for extended support)

You can add additional weights and styles:

- `Sk-Modernist-Light.woff2` (weight: 300)
- `Sk-Modernist-Medium.woff2` (weight: 500)
- `Sk-Modernist-SemiBold.woff2` (weight: 600)
- `Sk-Modernist-Italic.woff2` (weight: 400, style: italic)
- `Sk-Modernist-BoldItalic.woff2` (weight: 700, style: italic)

## Format

WOFF2 format is recommended for optimal performance and modern browser support.
If WOFF2 is not available, you can use TTF or OTF formats, but update the file extensions in `src/app/layout.tsx` accordingly.
